# Wolf Sheep model sample data: simdesign eFast

The dataset contains a complete nl object. The nl object has been used
to setup and run an eFast simulation design. It also contains the model
outputs within the simdesign object. Further analysis output can be
derived by submitting the dataset to analyze_nl().

## Usage

``` r
nl_eFast
```

## Format

nl object with defined experiment, simdesign and model output
