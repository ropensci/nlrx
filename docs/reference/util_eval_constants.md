# Evaluate if constants list of an experiment object is empty

Evaluate if constants list of an experiment object is empty

## Usage

``` r
util_eval_constants(nl)
```

## Arguments

- nl:

  nl object util_eval_constants checks if the constants list of an
  experiment within a nl object is empty. It reports an error message if
  no constants were defined. This evaluation is only done for
  [simdesign_simple](https://docs.ropensci.org/nlrx/reference/simdesign_simple.md).
