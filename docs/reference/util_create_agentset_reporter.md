# Create turtle/patches/links owns reporter

The reporter is used to measure agent variables in NetLogo and parse
them to a readable format

## Usage

``` r
util_create_agentset_reporter(metrics, breed)
```

## Arguments

- metrics:

  character vector, the names of variables to measure

- breed:

  character, the corresponding breed/agentset (e.g. "turtles",
  "patches", "links", "wolves", ...)
